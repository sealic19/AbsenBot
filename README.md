# AbsentBOT-siakad
build BOT with [Puppeteer](https://pptr.dev/) & [Telegram-bot-api](https://github.com/yagop/node-telegram-bot-api)

## Tools:
1. Puppeteer
2. Telegram-bot-api
3. VSCode

## For development
1. clone this repo
2. setup account
3. run ```yarn or npm install``` for install dependency;
4. run ```npm start``` and enjoy!
